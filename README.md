# E-Learning Cloud Platform

## Deskripsi
Platform pembelajaran online berbasis cloud menggunakan AWS.

## Arsitektur
- EC2 (backend)
- RDS (database)
- S3 (storage)
- CloudFront (CDN)
- Load Balancer

## Fitur
- Login/Register
- Upload materi
- Streaming video
